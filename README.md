# Elemental Deck Defense

Deck-building Roguelite Tower Defense Game.

## 開発

```bash
npm install
npm run dev
```

## GitHub Pages で配信する

1. リポジトリの **Settings** → **Pages**
2. **Source** で **GitHub Actions** を選択
3. `main` ブランチに push すると自動でビルド・デプロイされる

公開 URL: `https://<あなたのユーザー名>.github.io/elemental-deck-defense/`

> リポジトリ名を変えた場合は `vite.config.ts` の `repoName` を同じ名前にしてください。
